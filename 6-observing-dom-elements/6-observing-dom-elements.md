# DOM 엘리먼트 감시

Browser가 DOM el을 감시할 목적으로 제공하는 3개의 옵저버:

* `MutationObserver`: 자식 el, attr, text content의 변화를 감지
* `IntersectionObserver`: 크기의 변경을 감시
* `ResizeObserver`: viewport에 대한 상대적 위치를 감시
